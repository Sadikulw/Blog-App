## ✨ Features

- 🔐 User Authentication (JWT + Cookies)
- 🧑‍💻 Register & Login
- ✍️ Create, Edit, Delete Blogs
- 📖 View all blogs
- ⚡ Fast API (Express)
- 🎨 Responsive UI (Tailwind CSS)
